# Library Management System

A simple Library Management System built using HTML, CSS, JavaScript, and Firebase.

## Features

- Add books
- Search books
- Show all books
- Borrow books
- View borrowed books
- Admin page

## Tech Stack

- HTML
- CSS
- JavaScript
- Firebase Firestore

## Future Improvements

- Authentication
- Return books system
- Availability tracking
- Better UI
- Responsive design
