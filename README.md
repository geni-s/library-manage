# Library Management System 📚

A simple and beginner-friendly Library Management System built using HTML, CSS, JavaScript, and Firebase Firestore.

This project allows users to browse books, search books, borrow books, and return books while automatically managing book quantities in real time.

---

## Features 🚀

- Add books
- Show all books
- Search books
- Filter books
- Borrow books
- Return books
- View borrowed books
- Quantity tracking
- Admin page
- Firebase Firestore integration
- Dynamic book cards

---

## Tech Stack 🛠️

- HTML
- CSS
- JavaScript
- Firebase Firestore

---

## Firebase Features Used 🔥

- addDoc()
- getDocs()
- updateDoc()
- deleteDoc()
- getDoc()

---

## How It Works ⚡

### Borrow Book
- User clicks the Borrow button
- Book data gets stored in `borrowedBooks`
- Quantity decreases automatically

### Return Book
- User clicks the Return button
- Quantity increases automatically
- Borrowed entry gets deleted
